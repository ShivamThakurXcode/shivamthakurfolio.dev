FAQ

Do you design in Figma or in code?

Both. I usually design in Figma to explore layout and flow quickly, then translate it faithfully into responsive, accessible code. Because I also develop, the design never drifts from what is actually buildable.

Can you work from my brand guidelines?

Absolutely. If you have a brand, I design within it and extend it into a coherent web design system. If you do not, I can establish the core visual language — type, colour, spacing and components — as part of the work.

Will the design be responsive and accessible?

Yes. Every design is built mobile-first and tested across screen sizes, with sensible colour contrast, keyboard navigation and semantic structure so it works for everyone.
