Prodigichat

A SaaS WhatsApp marketing and automation tool — broadcast campaigns, chatbot flows and a shared team inbox, built on the official WhatsApp Cloud API.

Overview

Prodigichat lets businesses run their WhatsApp marketing from one place: send broadcast campaigns to opted-in contacts, automate replies with visual chatbot flows, and manage every incoming conversation from a shared team inbox — all on the official WhatsApp Cloud API.

The problem

WhatsApp is where customers actually reply, but the raw API is not something a marketing team can use directly. Businesses needed a friendly way to send campaigns at scale, automate common questions, and let a whole team answer from one number — without hitting rate limits or losing messages.

The solution

I built Prodigichat as a full-stack SaaS tool. A React dashboard drives campaign creation, a drag-and-drop chatbot flow builder, and a live team inbox. A Node.js API talks to the WhatsApp Cloud API and receives inbound messages via webhooks. Broadcasts are queued in Redis and sent by background workers that respect WhatsApp’s rate limits and retry failures, while WebSockets keep the shared inbox updating in real time for every agent.

Campaigns, automation and a shared inbox on one WhatsApp number — queued, rate-limit-safe, and updating in real time.

FAQ

What is Prodigichat?

Prodigichat is a SaaS WhatsApp marketing and automation tool. Businesses use it to send broadcast campaigns, build automated chatbot flows, and handle incoming conversations from a shared team inbox — all on top of the official WhatsApp Cloud API.

How does Prodigichat handle large broadcast campaigns?

Messages are pushed onto a Redis-backed job queue and sent by background workers that respect WhatsApp's rate limits. This keeps the dashboard responsive, retries failures, and lets a campaign to thousands of contacts run reliably without blocking the app.

What stack was used to build Prodigichat?

A React dashboard, a Node.js API, Redis for queues and real-time state, WebSockets for the live team inbox, and the WhatsApp Cloud API for sending and receiving messages via webhooks.
